## Overview

<a href="#">Image Resizer</a> addresses the workflow gap between full image editors and the need to simply make images smaller, larger, or appropriately sized for specific destinations. Photographers preparing images for web galleries, developers creating app icon sets, and content creators sizing images for social media platforms all need consistent resizing without the overhead of Photoshop or Preview for each file.

The <a href="#">preset system</a> stores frequently used size configurations for quick application. Define presets for common output requirements — social media image sizes, email attachment limits, web gallery dimensions, or specific client delivery specifications. Apply saved presets from a dropdown rather than re-entering dimensions for recurring tasks. <a href="#">Multiple preset application</a> generates several output sizes from each source image simultaneously, producing all required sizes in a single operation.

<a href="#">Finder integration</a> adds resizing actions to the right-click context menu for images and folders in Finder. Resize without opening the main application window for quick single-file operations. <a href="#">Drag-and-drop processing</a> accepts images and folders dropped directly onto the application window for immediate processing without file picker navigation.

The <a href="#">preview function</a> shows the expected output appearance and file size before committing to batch processing, allowing quality and size verification before the full batch runs. <a href="#">Processing progress display</a> shows completion status during large batch operations with file count and estimated time remaining. Export reports summarize what was processed, output sizes, and any errors encountered during batch operations. The combination of flexible dimension specification, format conversion, batch capability, and Finder integration makes Image Resizer the practical tool that stays in the Dock for <a href="#">frequent image preparation workflows</a> rather than requiring a full image editor launch for simple sizing tasks.

## Key Features

- <a href="#">Flexible resize targeting</a> by exact pixels percentage maximum dimension or output file size
- <a href="#">Proportional scaling</a> maintains correct aspect ratio across any mixed collection of source images
- <a href="#">Batch folder handling</a> processes complete image directories with consistent output settings applied
- <a href="#">Original file protection</a> through separate destination output with optional subfolder replication
- <a href="#">Cross-format conversion</a> resizes and converts between JPEG PNG WEBP and other formats simultaneously
- <a href="#">Quality level adjustment</a> for lossy formats precisely controls the size versus fidelity tradeoff
- <a href="#">Named output presets</a> apply saved size and format configurations instantly for recurring tasks
- <a href="#">Simultaneous multi-size output</a> produces all required image dimensions from each source in one pass
- <a href="#">Drag-and-drop processing</a> accepts images and folders on the window for immediate batch operations
